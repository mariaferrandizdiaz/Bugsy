Funciones y Procedimientos

¿Qué es la programación modular?

La programación modular consiste en dividir un programa grande en partes más pequeñas llamadas funciones o subprogramas. Estas partes realizan tareas específicas y ayudan a que el código sea más fácil de leer, entender, probar y reutilizar.

¿Qué es una función?

Una función es un bloque de código que puede ser llamado (ejecutado) desde otra parte del programa. Puede recibir datos de entrada llamados parámetros y puede devolver un valor como resultado.

Ejemplo de una función simple:


int sumar(int a, int b) {
    return a + b;
}

Parámetros formales y variables locales

  • Parámetros formales: son los que recibe la función cuando es llamada (por ejemplo a y b en el ejemplo anterior).
  • Variables locales: son variables que se declaran dentro de la función y solo existen mientras esa función se ejecuta.

Ventajas de la programación modular

  • Permite reutilizar código fácilmente.
  • Facilita encontrar y corregir errores.
  • Permite trabajar en equipo: cada persona puede trabajar en una función distinta.
  • Mejora la legibilidad del código.

¿Cómo se pasan los datos a una función?

En C, los datos se pueden pasar de dos formas:

1. Paso por valor (por defecto):

Se hace una copia del valor. Si lo cambias dentro de la función, no afecta al original.


void cambiar(int x) {
    x = 10;
}

int main() {
    int valor = 5;
    cambiar(valor);
    printf("%d", valor); // Imprime 5, no cambia
}

2. Paso por referencia (con punteros):

Se pasa la dirección de memoria de la variable. Si se cambia dentro de la función, también cambia fuera.


void cambiar(int *x) {
    *x = 10;
}

int main() {
    int valor = 5;
    cambiar(&valor);
    printf("%d", valor); // Imprime 10, sí cambia
}

Uso de const en funciones

Cuando quieres asegurarte de que una variable no será modificada dentro de una función, puedes usar const:


void imprimir(const int numero) {
    printf("%d", numero);  // No puede ser modificado
}

Prototipos de funciones

Cuando defines una función más abajo en el código, necesitas escribir antes su prototipo:


int sumar(int, int);  // Prototipo

int main() {
    printf("%d", sumar(3, 4));
}

int sumar(int a, int b) {
    return a + b;
}

Funciones que no devuelven nada: void

Cuando una función no necesita devolver nada, se usa el tipo void:


void saludar() {
    printf("¡Hola mundo!");
}

Pasar arrays a funciones

Los arrays se pasan por referencia automáticamente. Si quieres evitar que se modifiquen, usa const.


void imprimirArray(const int datos[], int tam) {
    for (int i = 0; i < tam; i++) {
        printf("%d ", datos[i]);
    }
}

Resumen del paso de parámetros

Tipo de parámetro Se modifica dentro de la función Afecta al valor original
int x Sí No
const int x No No
int *x Sí Sí

Comprender cómo funcionan las funciones en C te permitirá escribir programas más limpios, claros y reutilizables.
