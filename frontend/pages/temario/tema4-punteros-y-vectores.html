Punteros y Vectores

¿Qué es un puntero?

Un puntero en C es una variable que almacena la dirección de memoria de otra variable. En lugar de guardar un valor directamente, guarda dónde está ese valor en la memoria.

Esto es útil para manipular directamente la memoria, trabajar con arrays, funciones, estructuras y optimizar el rendimiento.

Declaración básica

int *ptr;  // Puntero a un entero

El operador * indica que se está declarando un puntero, no un valor entero común.

¿Cómo funcionan los punteros?

Supón que tienes una variable int x = 5;. Si quieres crear un puntero que apunte a esa variable, usas el operador & para obtener su dirección:


int x = 5;
int *ptr = &x;  // ptr guarda la dirección de x

Y si quieres acceder al valor al que apunta el puntero, usas * (operador de desreferenciación):


printf("%d", *ptr);  // imprime el valor de x (5)

Precauciones importantes

  • No uses punteros sin inicializar: pueden apuntar a zonas de memoria no válidas.
  • Siempre es buena práctica inicializarlos con NULL si aún no tienen destino.
  • Evita modificar punteros con operaciones aritméticas complejas sin entender su efecto.

Ejemplo: intercambio de valores con punteros


void intercambiar(int *a, int *b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}

int main() {
    int x = 10, y = 20;
    intercambiar(&x, &y);
    printf("x = %d, y = %d\\n", x, y);  // x = 20, y = 10
}

Vectores (arrays) en C

Un vector es una estructura que permite almacenar múltiples valores del mismo tipo bajo un mismo nombre. Cada valor se accede mediante un índice.

int notas[5] = {8, 6, 7, 9, 10};

El primer elemento es notas[0] y el último notas[4]. El índice siempre comienza en 0.

Acceso con punteros

El nombre de un vector es en sí mismo un puntero al primer elemento.


int notas[3] = {5, 7, 9};
int *p = notas;
printf("%d", *(p + 1));  // imprime 7

Esto significa que puedes recorrer un array tanto con índices como con punteros:


for (int i = 0; i < 3; i++) {
    printf("%d ", *(notas + i));
}

Cadenas de caracteres (strings)

Una cadena es un array de caracteres terminado en el carácter nulo '\\0'.


char saludo[] = "Hola";  // equivalente a {'H','o','l','a','\\0'}
printf("%s", saludo);    // imprime: Hola

Arrays multidimensionales

Un array de dos dimensiones se interpreta como una matriz (filas y columnas):


int matriz[2][3] = {
    {1, 2, 3},
    {4, 5, 6}
};

printf("%d", matriz[1][2]);  // imprime 6

Resumen final

  • Un puntero guarda una dirección de memoria
  • Se usa * para declarar y acceder al contenido apuntado
  • Los arrays son bloques contiguos de memoria: se acceden con índices o punteros
  • Las cadenas en C terminan con '\\0'
  • Con punteros puedes manipular directamente el contenido de variables o recorrer vectores

Practicar con punteros ayuda a comprender mejor la gestión de memoria en C y optimizar los programas.
