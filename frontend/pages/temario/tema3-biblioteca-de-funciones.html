Biblioteca de funciones en C

¿Qué es una biblioteca en C?

En el lenguaje C, una biblioteca es un conjunto de funciones ya escritas y disponibles para que el programador las utilice sin tener que definirlas desde cero. Estas funciones están organizadas por temas o funcionalidades, y se incluyen en los programas mediante la instrucción #include.

Biblioteca <stdio.h>: Entrada y salida de datos

La biblioteca estándar de entrada y salida (stdio.h) contiene las funciones necesarias para mostrar datos al usuario (salida) y para recibir datos del usuario (entrada).

Funciones principales

  • printf(): muestra información en la consola.
  • scanf(): lee valores introducidos por el usuario.
  • getchar(): lee un carácter.
  • putchar(): escribe un carácter.
  • gets(): lee una cadena de caracteres (ya no recomendada por seguridad).
  • puts(): escribe una cadena de caracteres.

Ejemplo con printf y scanf

#include <stdio.h>

int main() {
    char nombre[20];
    int edad;

    printf("Introduce tu nombre: ");
    scanf("%s", nombre);

    printf("Introduce tu edad: ");
    scanf("%d", &edad);

    printf("Hola %s, tienes %d años.\\n", nombre, edad);
    return 0;
}

Formatos más comunes en printf

  • %d o %i: entero decimal
  • %o: entero en octal
  • %x: entero en hexadecimal
  • %u: entero sin signo
  • %f: número con coma flotante
  • %e: notación científica
  • %s: cadena de caracteres
  • %c: un solo carácter
  • %%: el carácter % literal

Secuencias de escape útiles

Se utilizan para representar caracteres especiales:

  • \\n: salto de línea
  • \\t: tabulador horizontal
  • \\b: retroceso
  • \\r: retorno de carro
  • \\a: alerta sonora
  • \\\": comilla doble
  • \\': comilla simple
  • \\\\: barra invertida
  • \\?: signo de interrogación
  • \\xhh: carácter en hexadecimal

Biblioteca <math.h>: Funciones matemáticas

Contiene funciones para realizar operaciones matemáticas avanzadas. Requiere incluir <math.h> y, en muchos compiladores como GCC, usar -lm al compilar.

Funciones frecuentes en math.h

  • double sin(x), cos(x), tan(x): trigonométricas básicas
  • double asin(x), acos(x), atan(x): trigonométricas inversas
  • double exp(x): exponencial de x
  • double log(x): logaritmo natural
  • double log10(x): logaritmo base 10
  • double pow(x, y): potencia (x^y)
  • double sqrt(x): raíz cuadrada
  • double fabs(x): valor absoluto

Ejemplo de uso de math.h

#include <stdio.h>
#include <math.h>

int main() {
    double base = 2.0, exponente = 3.0;
    double resultado = pow(base, exponente);

    printf("2 elevado a 3 es: %f\\n", resultado);

    return 0;
}

Recomendaciones

  • Incluye siempre #include <math.h> si usas funciones matemáticas.
  • Compila con -lm si es necesario (por ejemplo: gcc archivo.c -lm).
  • Recuerda que muchas funciones trabajan con double por defecto.

Estas bibliotecas amplían enormemente lo que puedes hacer con C y son fundamentales para desarrollar programas interactivos y con lógica matemática.
