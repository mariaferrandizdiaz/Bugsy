Introducción a la Programación en C

1. ¿Cómo se resuelve un problema con un programa?

Antes de poder escribir código, necesitamos entender el problema que queremos resolver y seguir una serie de pasos:

  1. Definir o analizar el problema: ¿Qué se necesita resolver?
  2. Diseñar un algoritmo: Plantear una solución paso a paso.
  3. Convertir el algoritmo en un programa: Usar un lenguaje como C.
  4. Ejecutar y comprobar que funciona: Validar que la solución sea correcta.

En resumen: Problema → Algoritmo → Programa.

2. ¿Qué es un algoritmo?

Un algoritmo es una serie de pasos claros y ordenados para resolver un problema. No lo resuelve por sí mismo, pero indica cómo hacerlo. Solo puede ejecutarse si alguien (un procesador o computadora) entiende sus instrucciones.

2.1. ¿Y si el algoritmo es para humanos?

Nosotros podemos entender instrucciones más ambiguas, pero los ordenadores necesitan lenguajes exactos. Por eso usamos lenguajes de programación y, antes de eso, a veces pseudocódigo, que es como un lenguaje intermedio más fácil de leer.

3. Herramientas que nos ayudan a programar

  • Diagramas de flujo: dibujan los pasos como cajas y flechas.
  • Diagramas N-S (Nassi-Schneiderman): parecidos a los de flujo, pero con otra estructura.
  • Pseudocódigo: escribir la lógica del programa sin usar un lenguaje formal.

4. ¿Qué contiene un diagrama de flujo?

  • Procesos (acciones)
  • Entradas y salidas
  • Decisiones (como preguntas de sí/no)
  • Flechas que conectan los pasos
  • Conectores para seguir en otras páginas

5. ¿Qué es una variable?

Es un espacio en memoria que guarda un dato. Ese dato puede cambiar a lo largo del programa, por eso se llama "variable".

  • Nombre: cómo la llamamos en el programa.
  • Tipo: qué clase de dato guarda (número, texto, etc.).
  • Valor: el dato que contiene en ese momento.

Tipos de variables comunes:

  • Numéricos: enteros o decimales.
  • Carácter: un solo símbolo, como 'A' o '9'.
  • Ristras: cadenas de texto como "Hola Mundo".
  • Lógicos: verdadero o falso.

6. Cómo guardar y mostrar datos

Algunos ejemplos:

i = 30;
X = i / j;
A = "La Laguna";
F = Verdadero;
Leer i;
Escribir "El resultado es:", A;

Para declarar variables en C:

int A, B, C;
float R;

7. ¿Cómo toma decisiones un programa?

Con estructuras como:

if (condición):

if (a > b) {
    max = a;
} else {
    max = b;
}

while: mientras se cumpla una condición

while (i < 10) {
    printf("%d", i);
    i++;
}

do-while: se ejecuta al menos una vez

int i = 0;
do {
    printf("%d", i);
    i++;
} while (i < 10);

for: para un número de repeticiones conocido

for (int i = 0; i < 10; i++) {
    printf("%d", i);
}

switch: elegir entre varias opciones

switch(opcion) {
    case 1: z = x + y; break;
    case 2: z = x - y; break;
    case 3: z = x * y; break;
    case 4: z = (float)x / y; break;
}

Estas estructuras son la base para que los programas puedan tomar decisiones y repetir acciones.
