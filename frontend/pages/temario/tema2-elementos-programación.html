Identificadores, Tipos de Datos, Variables y Constantes

¿Qué es un identificador?

Un identificador es el nombre que damos a cosas como variables, funciones o constantes. Por ejemplo, edad o mostrarMensaje son identificadores.

En C, los identificadores:

  • Distinguen entre mayúsculas y minúsculas (Edad ≠ edad).
  • No pueden comenzar por un número.
  • Solo pueden contener letras, números y guiones bajos (_).
  • No pueden ser palabras reservadas como int, return o while.

Tipos de datos básicos

En C, cada variable debe indicar qué tipo de datos va a guardar. Los principales tipos son:

  • int: Números enteros.
  • float: Números decimales.
  • char: Caracteres individuales como 'A' o '7'.
  • double: Decimales más precisos que float.
  • void: Sin tipo (se usa en funciones que no devuelven nada).

Modificadores de tipo

Permiten ajustar el tamaño o el comportamiento del tipo de dato. Ejemplos:

  • unsigned int: Entero sin signo (no admite negativos).
  • long double: Número decimal con mucho rango.
  • short int: Entero pequeño.

Modificadores de acceso

  • const: La variable no puede cambiar.
  • volatile: La variable puede cambiar desde fuera del programa (ej. reloj del sistema).

¿Dónde se pueden declarar las variables?

  • Globales: Fuera de todas las funciones.
  • Locales: Dentro de una función.
  • Parámetros: Dentro de la cabecera de una función.
  • Bloques: Dentro de un bloque entre llaves { }.

Especificadores de almacenamiento

  • auto: Local y temporal (por defecto).
  • extern: Variable declarada en otro módulo.
  • static: Conserva el valor entre ejecuciones de la función.
  • register: Intenta guardar la variable en un registro rápido del procesador.

Constantes

Las constantes son valores que no cambian durante la ejecución del programa. Ejemplos:

  • const int max = 100;
  • Constantes octales: 012, 073
  • Constantes hexadecimales: 0x1F, 0xFF
  • Constantes de cadena: "Hola mundo"
  • Caracteres especiales como \\n (salto de línea), \\t (tabulación)

Operadores

Los operadores permiten realizar cálculos o comparar valores. Algunos ejemplos:

  • Aritméticos: +, -, *, /, %
  • Relacionales: ==, !=, >, <
  • Lógicos: &&, ||, !
  • Asignación: =, +=, *=, etc.

Estructura básica de un programa en C

#include <stdio.h>

int main() {
    // Declaraciones
    int edad = 0;
    char nombre[50];

    // Entrada de datos
    printf("Introduce tu nombre: ");
    scanf("%s", nombre);

    printf("Introduce tu edad: ");
    scanf("%d", &edad);

    // Condicional
    if (edad >= 18) {
        printf("%s es mayor de edad.\\n", nombre);
    } else {
        printf("%s es menor de edad.\\n", nombre);
    }

    return 0;
}

Buenas prácticas y estilo

  • Comenta tu código para que otros lo entiendan.
  • Usa nombres de variables claros y significativos.
  • No escribas expresiones demasiado complicadas.
  • Escribe el código de forma ordenada e indentada.
  • Evita errores comunes como confundir = con ==.
